# Dcat-admin-extension 文件管理

## 环境
 - PHP >= 7.1.0
 - Dcat-admin ^2.0


## 安装

### composer 安装

```
composer require xingchenboy/dcat-files-management
```

### 安装扩展

在 `开发工具->扩展` 安装本扩展

### 访问
在 `菜单` 里添加 media 进行访问

## 开源协议

*  本扩展遵循 `Apache2` 开源协议发布，提供个人及商业免费使用。
